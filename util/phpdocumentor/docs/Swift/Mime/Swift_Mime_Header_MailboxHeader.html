getMailboxes (line 276)

Get all mailboxes in this Header as key=>value pairs.

The key is the address and the value is the name (or null if none set). Example:

  1.  <?php
  2.  $header = new Swift_Mime_Header_MailboxHeader('From',
  3.   array('chris@swiftmailer.org' => 'Chris Corbyn',
  4.   'mark@swiftmailer.org' => 'Mark Corbyn')
  5.   );
  6.  print_r($header->getMailboxes());
  7.  // array (
  8.  // chris@swiftmailer.org => Chris Corbyn,
  9.  // mark@swiftmailer.org => Mark Corbyn
  10.  // )
  11.   ?>

setValue (line 368)

Set the value of this Header as a string.

The tokens in the string MUST comply with RFC 2822, 3.6.2. The value will be parsed so getMailboxes() and other related methods return appropriate values. This can be useful if working with raw data from an email not generated by Swift. Example:

  1.  <?php
  2.  //Sets two mailboxes is the Header.
  3.   $header->setValue('Person Name <person@address.com>,
  4.   (This is a comment) "Person 2\\, BSc\\." <person2@address.com>'
  5.   );
  6.  ?>
